# 播放器

<Callout type="warning" emoji="⚠️">

Player 组件用于播放故事

> 注意：您必须将 `Player` 组件包裹在 [GameProviders](./game-providers) 组件中以提供游戏实例

如果您正在使用抽象类创建故事，请查看 [Core](/documentation/core) 文档

## 示例

这是一个如何使用 `Player` 组件的基本示例：

```tsx
"use client";

import { GameProviders, Player } from "narraleaf-react";
import { useState } from "react";
import { story } from "./my-story";

export default function Page() {
    return (
        <GameProviders>
            <Player story={story} onReady={({ liveGame }) => {
                liveGame.newGame();
            }}
                width="100vw"
                height="100vh"
            />
        </GameProviders>
    );
}
```

## Props

### story

您要播放的故事对象

### width

播放器的宽度

### height

播放器的高度

### className

播放器的类名

### onReady

当播放器准备就绪时调用的回调函数。它返回 [ScriptCtx](../core/types/ScriptCtx) 对象

### onEnd

当播放器结束时调用

此回调在故事结束时调用

### children

播放器的子组件

您可以将子组件传递给播放器

例如，您可以使用 [PageRouter API](./page-router) 创建自己的快捷菜单
