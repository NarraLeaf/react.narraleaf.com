# 路由器

Router 用于在页面之间导航。

```ts
import { useRouter } from "narraleaf-react";
```

```tsx
const router = useRouter();
```

## 公共方法

### push


通过页面的 id 导航到一个页面。

```ts
router.push("about");
```

游戏将导航到 id 为 `about` 的页面。

```ts
<Page id="about">
    {/* 这些元素将被显示 */}
    {/* 你的内容 */}
</Page>
```

- `id: string` - 页面的 id。
- 返回 `this`

### back

导航到上一个页面。

如果没有上一个页面，此方法将不执行任何操作。

- 返回 `this`

### forward

导航到下一页。

如果没有下一页，此方法将不执行任何操作。

- 返回 `this`

### cleanHistory

清除导航历史。

- 返回 `this`

### getCurrentId

获取当前页面的 id。

- 返回 `string`
