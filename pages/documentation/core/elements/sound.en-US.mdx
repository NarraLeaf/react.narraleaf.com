Sound controls the sound of the game

## Static Method

### voice

<Steps>

Create a voice sound

```ts
const voice = Sound.voice({
    src: 'voice.mp3',
});

// is equivalent to
const voice = new Sound({
    src: 'voice.mp3',
    type: SoundType.Voice,
});
```

- `arg0: Partial<ISoundUserConfig> | string` - The config of the sound or the source of the sound, see [ISoundUserConfig](../types/ISoundUserConfig)
- Returns `Sound`

</Steps>

## Chainable Method

### play

<Steps>

Play the sound and wait for it to finish

> **Note**: Sounds marked as `Bgm` cannot be played directly.

```ts
const sound = new Sound({
    src: 'sound.mp3',
});
scene.action([
    sound.play(1000), // play and fade in 1 second
]);
```

- `duration?: number` - Fade duration in milliseconds

</Steps>

### stop

<Steps>

Stop the sound. After stopping, the sound will go back to the beginning

- `duration?: number` - Fade duration in milliseconds

</Steps>

### fade

<Steps>

Fade the sound

- `start: number | undefined` - Start volume, from 0 to 1, default is current volume
- `end: number` - End volume, from 0 to 1
- `duration: number` - Duration in milliseconds

```ts
const sound = new Sound({
    src: 'sound.mp3',
    volume: 0,
});
scene.action([
    sound
        .play()
        .fade(0, 1, 1000), // fade in 1 second
]);
```

</Steps>
