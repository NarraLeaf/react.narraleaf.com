# 图片

图片对于视觉小说非常重要。它用于显示角色的外观。

在NarraLeaf-React中，图片仅控制角色的外观。它不控制角色的对话。为此，您可以使用[Character](../character)。

## 公共方法

### constructor

- `name: string` - 图片的名称，它将用于调试目的
- `config?: DeepPartial<ImageConfig>` - [ImageConfig](../types/ImageConfig)

### dispose

通常，**您不需要调用此方法**。此方法用于停用图像。  
停用后的图像将不会在剩余的故事中显示在屏幕上。

### setSrc

- `src: string | StaticImageData` - 图像的来源
- `transition?: ITransition` - 有关ITransition，请参见[过渡](../types/ITransition)

```typescript
image.setSrc("your/image/src", new Fade(1000));
```

### applyTransform

- `transform: Transform` - [变换](../animation/transform)

### show

显示图片。

#### 重载 1 / 3

立即显示图片。

```typescript
image.show();
```

#### 重载 2 / 3

使用自定义变换显示图像。

> **注意：**您必须在变换属性中添加`alpha: 1`以显示图像。

- `options: Transform` - [变换](../animation/transform)

#### 重载 3 / 3

使用变换配置显示图像。一个带有alpha:1的变换将应用于图像。

- `options: Partial<TransformDefinitions.CommonTransformProps>` - 见 TransformDefinitions.[CommonTransformProps](../types/CommonTransformProps)

```typescript
image.show({
    duration: 1000,
    ease: "easeInOut",
});
```

### hide

隐藏图片。

#### 重载 1 / 3

立即隐藏图片。

```typescript
image.hide();
```

#### 重载 2 / 3

立即使用自定义变换隐藏图片。

> **注意：**您必须在变换属性中添加`alpha: 0`以隐藏图像。

- `options: Transform` - [变换](../animation/transform)

#### 重载 3 / 3

使用变换配置隐藏图像。一个带有alpha:0的变换将应用于图像。

- `options: Partial<TransformDefinitions.CommonTransformProps>` - 见 TransformDefinitions.[CommonTransformProps](../types/CommonTransformProps)

```typescript
image.hide({
    duration: 1000,
    ease: "easeInOut",
});
```

### copy

复制图像。

```typescript
const newImage = image.copy();
```

### addWearable

Add a wearable to the image

> Wearable is a feature that allows you to add an image that related to the main image. For example, you can add a hat to the character.
> And that wearable image will move/scale/animate with the main image.

- `children: Image | Image[]` - Wearable image or images
- Returns `this`

```typescript
const childImage = new Image(/* ... */);

image.addWearable(childImage);
```

### bindWearable

Bind this image to a parent image as a wearable

- `parent: Image` - Parent image
- Returns `this`

```typescript
childImage.bindWearable(parentImage);
```
