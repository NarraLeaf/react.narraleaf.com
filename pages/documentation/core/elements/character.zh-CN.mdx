角色控制角色的对话。

**注意：** `Character` 本身不驱动舞台立绘布局；舞台展示请使用 [Image](./image)。**对话框旁小头像**（ADV 头像）通过 `avatar` / `portraits` 配置，并在 UI 中使用 [`Avatar`](../../player/dialog) — 详见 [Dialog](../../player/dialog)。

## 公共方法

### constructor

<Steps>

- `name: string | null` - 如果为 null，则为旁白。旁白在对话框中不显示任何名字。
- `config?: DeepPartial<CharacterConfig>` - [CharacterConfig](../types/CharacterConfig)

</Steps>

### [[Callable]]

<Steps>

`character.say` 可以作为标签函数使用。

```typescript
character`Hello, ${Word.color("Alice", "#f00")}!`, // will output "Hello, Alice" with "Alice" in red color

// 等价于

character.say`Hello, ${Word.color("Alice", "#f00")}!`
```

> **注意：** 使用一些打包工具，如 Webpack，你不能像这样链式调用这个方法：
> 
> ```typescript
> scene.action([
>     character`sentence 1` `sentence 2`, // 这会抛出一个错误
> ]);
> ```

</Steps>

## 链式方法

### say

<Steps>

#### 重载 1 / 4

说一句话。

```typescript
character
    .say("Good morning!")
    .say("How are you?")
```

- `content: string` - 句子的内容
- `config?: SentenceUserConfig` - [SentenceUserConfig](../types/SentenceUserConfig)

#### 重载 2 / 4

使用自定义句子对象。

```typescript
character.say(
    new Sentence(character, [
        "早上好，我是 ",
        new Word("Alice", {color: "#f00"}), // 一些单词可以着色
    ])
) // // 将输出 "早上好，我是 Alice，其中 "Alice" 为红色
```

> **注意：** 对话的名称将与句子的名称相同。

- `content: Sentence` - 参见 [Sentence](./character/sentence)

#### 重载 3 / 4

使用字符串和 Word 对象的混合内容。

```typescript
character.say([
    "你好，",
    new Word("艾丽斯", {color: "#f00"}), // 一些单词可以着色
]) // 将输出 "你好，艾丽斯"，其中 "艾丽斯" 为红色
```

- `content: SentencePrompt` - 参见 [SentencePrompt](../types/SentencePrompt)
- `config?: SentenceUserConfig` - [SentenceUserConfig](../types/SentenceUserConfig)

#### 重载 4 / 4

使用 SentencePrompt 的简写形式。

```typescript
    character.say`你好，${Word.color("艾丽斯", "#f00")}！` // 将输出 "你好，艾丽斯"，其中 "艾丽斯" 为红色
```

- `texts: TemplateStringsArray` - 模板字符串数组
- `...words: SingleWord[]` - [SingleWord](../types/SentencePrompt#SingleWord)

</Steps>

### setName

<Steps>

    设置角色的名称。

```typescript
character
    .setName("艾丽斯（生气）")
    .say("你想要什么？")
```

- `name: string` - 角色的名称

</Steps>

### setAvatar

<Steps>

设置或清除**角色级**对话框头像（画外音、以及无可见绑定立绘时的回退）。

```typescript
const alice = new Character("Alice")
    .setAvatar("/assets/alice/avatar-default.png");

alice.setAvatar(false); // 关闭角色级头像
```

- `avatar: DialogAvatar | false | null` — 图片源、解析器、`false` 关闭，或 `null` 清除。

解析语义与优先级见 [Dialog](../../player/dialog)。

</Steps>

### addPortrait

<Steps>

将舞台 [`Image`](./image) 绑定到该角色，并可选择为该绑定配置**立绘级**对话框头像（字符串或解析器）。

```typescript
alice.addPortrait(aliceBody, {
    avatar: "/assets/alice/avatar-normal.png",
});
```

- `image: Image` — 舞台立绘。
- `config?: { avatar?: DialogAvatar }` — 该绑定可选的头像覆盖。

</Steps>

### setPortraits

<Steps>

替换完整的 `portraits` 列表（可为裸 `Image` 或 `{ image, avatar }`）。

```typescript
alice.setPortraits([
    { image: aliceBody, avatar: "/assets/alice/avatar-a.png" },
    bobPortraitImage,
]);
```

- `portraits: (Image | CharacterPortraitConfig)[]` — 见 [CharacterConfig](../types/CharacterConfig)。

</Steps>

### apply

<Steps>

这是一个 `say` 的特殊别名。

```typescript
character`Hello, ${Word.color("Alice", "#f00")}!`;
```

> **注意：** 使用一些打包工具，如 Webpack，你不能像这样链式调用这个方法：
> 
> ```typescript
> scene.action([
>     character`sentence 1` `sentence 2`,
> ]);
> ```
> 
> 但你可以像这样：
> 
> ```typescript
> scene.action([
>     character`sentence 1`.setName("Alice"),
> ]);
> ```

这个方法在角色只说一句话时缩短了代码。

```typescript
scene.action([
    character`Hello, ${Word.color("Alice", "#f00")}!`,
]);
```

- `texts: TemplateStringsArray` - 模板字符串数组
- `...words: SingleWord[]` - [SingleWord](../types/SentencePrompt#SingleWord)

</Steps>
