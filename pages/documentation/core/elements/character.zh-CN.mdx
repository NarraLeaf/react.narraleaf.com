角色控制角色的对话。

**注意：** 角色不控制角色的外观。为此，您可以使用 [Image](./image)。

## 公共方法

### constructor

- `name: string | null` - 如果为 null，则为旁白。旁白不在对话框中显示任何名称。
- `config?: DeepPartial<CharacterConfig>` - [CharacterConfig](../types/CharacterConfig)

### say

#### 重载 1 / 3

说一句话。

```typescript
character
    .say("Hello, world!")
    .say("How are you?")
```

- `content: string` - 句子的内容

#### 重载 2 / 3

使用自定义句子对象。

```typescript
character.say(
    new Sentence(character, [
        "你好, ",
        new Word("世界", {color: "#f00"}), // 有一些单词可以着色
    ])
) // 将输出“你好，世界”，“世界”为红色
```

> **注意：** 对话框的名称将与句子的名称相同。

- `content: Sentence` - 见 [句子](./character/sentence)

#### 重载 3 / 3

使用字符串和 Word 对象的混合内容。

```typescript
character.say([
    "你好, ",
    new Word("世界", {color: "#f00"}), // 有一些单词可以着色
]) // 将输出“你好，世界”，“世界”为红色
```

- `content: (string | Word)[]` - 见 [词语](./character/word)
