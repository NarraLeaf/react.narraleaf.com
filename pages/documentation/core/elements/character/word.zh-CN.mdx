# 词语

<Callout type="warning" emoji="⚠️">
    This page is under construction.
</Callout>

词语是句子的一部分。它用于为文本添加样式。

构建一个红色的词语：

```ts
import { Word } from 'narraleaf-react';

const word = new Word("你好", {color: "#f00"});
```
