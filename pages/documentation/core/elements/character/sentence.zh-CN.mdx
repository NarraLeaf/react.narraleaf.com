# 句子

句子是角色的一部分。它用于构建对话。

构建句子：

```ts
import { Sentence } from 'narraleaf-react';

const character = new Character("角色1");
const sentence = new Sentence(character, [
    "你好, ",
    new Word("世界", {color: "#f00"}), // 有一些单词可以着色
]);
```

or 

```ts
const sentence = new Sentence(character, "你好, 世界");
```

or

```ts
const sentence = new Sentence(character, [
    ({storable}) => `Hello, ${storable.getNamespace("game").get("playerName")}`
]);
```

## 公共方法

### constructor

- `character: Character | null` - 如果为 null，则为旁白。旁白不在对话框中显示任何名称。
- `content: SentencePrompt` - 见 [SentencePrompt](../../types/SentencePrompt)
- `config?: Partial<SentenceConfig>` - [SentenceConfig](../../types/SentenceConfig)

### copy

复制一个句子

```ts
const sentence = new Sentence(character, "Hello, world");
const sentence2 = sentence.copy();
```

- Returns: `Sentence`
