文本允许您在屏幕上显示文本。

例如：

```ts
const t1 = new Text("之后，发生了另一个故事...", {
    position: CommonPosition.Positions.Center,
    fontColor: "#f00",
    fontSize: 20,
});

// 将背景变为黑色并添加淡入效果
// 然后显示文本 "之后，发生了另一个故事..."
scene2.setBackground("#000", new Fade(1000));
t1
    .show({
        duration: 1000,
        ease: "easeInOut",
    });
```

### show

<Steps>

#### 重载 1 / 3

立即显示文本

```ts
text.show();
```

#### 重载 2 / 3

使用自定义变换显示文本

> **注意**: 您需要在属性中添加 "opacity: 1" 使文本可见

```ts
text.show(new Transform<TransformDefinitions.TextTransformProps>([
    {
        props: {
            opacity: 1,
        },
        options: {
            duration: 1000,
            ease: "easeInOut",
        },
    }
]));
```

- `options: Transform<TransformDefinitions.TextTransformProps>` - 参见 [Transform](../animation/transform)\<[TransformDefinitions](../types/TransformDefinitions).[TextTransformProps](../types/TransformDefinitions#TextTransformProps)>

#### 重载 3 / 3

使用变换配置显示文本

```ts
text.show({
    duration: 1000,
    ease: "easeInOut",
});
```

- `options: Partial<TransformDefinitions.CommonTransformProps>` - 参见 [TransformDefinitions](../types/TransformDefinitions).[CommonTransformProps](../types/TransformDefinitions#CommonTransformProps)

</Steps>

### hide

<Steps>

#### 重载 1 / 3

立即隐藏文本

```ts
text.hide();
```

#### 重载 2 / 3

使用自定义变换隐藏文本

> **注意：**: 您需要在属性中添加 "opacity: 0" 使文本不可见

```ts
text.hide(new Transform<TransformDefinitions.TextTransformProps>([
    {
        props: {
            opacity: 0,
        },
        options: {
            duration: 1000,
            ease: "easeInOut",
        },
    }
]));
```

- `options: Transform<TransformDefinitions.TextTransformProps>` - 参见 [Transform](../animation/transform)\<[TransformDefinitions](../types/TransformDefinitions).[TextTransformProps](../types/TransformDefinitions#TextTransformProps)>

#### 重载 3 / 3

使用变换配置隐藏文本

```ts
text.hide({
    duration: 1000,
    ease: "easeInOut",
});
```

- `options: Partial<TransformDefinitions.CommonTransformProps>` - 参见 [TransformDefinitions](../types/TransformDefinitions).[CommonTransformProps](../types/TransformDefinitions#CommonTransformProps)

</Steps>
