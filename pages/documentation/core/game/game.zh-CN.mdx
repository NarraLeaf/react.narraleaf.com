# Game

<Callout type="warning" emoji="⚠️">
    此页面正在建设中
</Callout>

Game 是代表游戏本身的主类。它包含游戏的所有配置和 [LiveGame](./live-game)。

## 公共属性

### preference

[Preference](./preference/preference) 实例

- `Preference<GamePreference>` - 参见 [GamePreference](../types/GamePreference)

```typescript
game.preference.setPreference("autoForward", true);
```

## 公共方法

### constructor

- `config: DeepPartial<GameConfig>` - 游戏的配置，参见 [GameConfig](../types/GameConfig)

### useComponent\<T extends keyof ComponentsTypes\>

关于 ComponentsTypes，参见 [ComponentsTypes](../types/ComponentsTypes)

- `key: T` - 要使用的组件的键
- `components: ComponentsTypes[T]` - 组件配置，参见 [ComponentsTypes](../types/ComponentsTypes)

### getLiveGame

- `return: LiveGame` - [LiveGame](./live-game) 实例

### configure

为游戏分配新的配置

```typescript
const { game } = useGame();

useEffect(() => {
    game.configure({
        app: {
            debug: true,
            logger: true,
        },
        player: {
            ratioUpdateInterval: 0, // 立即更新玩家大小

            /* 鼠标指针 */
            cursor: "cursor.jpeg",
            cursorHeight: 60,
            cursorWidth: 60,
        }
    });
}, []);
