# Namespace\<T extends NameSpaceContent\<keyof T\>\>

Namespace 是一个类，表示 [Storable](../storable) 的命名空间。

## 公共属性

### name

- `string` - 命名空间的可读名称

## 公共方法

### constructor

<Steps>

- `name: string` - 命名空间的可读名称
- `initContent: T` - 命名空间的初始内容
- `key?: string` - 命名空间的键，默认为名称

</Steps>

### set\<Key extends keyof T\>

<Steps>

示例：
```typescript
const namespace = new Namespace<{name: string}>('player1', {name: 'John Doe'});

namespace.set('name', 'Jane Smith');
```

- `key: Key` - 要设置的值的键
- `value: T[Key]` - 要设置的值
- 返回 `this`

</Steps>

### get\<Key extends keyof T = any\>

<Steps>

- `key: Key` - 要获取的值的键
- 返回 `T[Key]`

</Steps>

### equals\<Key extends keyof T = any\>

<Steps>

- `key: Key` - 要比较的值的键
- `value: T[Key]` - 要比较的值
- 返回 `boolean`

</Steps>

### assign

<Steps>

- `values: Partial<T>` - 要分配的内容
- 返回 `this`

</Steps>
