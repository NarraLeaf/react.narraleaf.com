# 淡入

> 此过渡效果专为[图像](../../elements/image)元素设计

从当前图像淡入到下一张图像  
下一张图像将淡入

## 用法

```tsx
import { FadeIn } from "narraleaf-react";

image.char("image1.jpg", new FadeIn(3000));
```

## 公共方法

### constructor

- `duration: number` - 过渡效果的持续时间，以毫秒为单位
- `startPos: [xOffset: number, yOffset: number]` - 图像的起始位置，默认为 `[0, 0]`。例如，`[50, 50]` 将使图像从右侧 50 像素和上方 50 像素的位置开始
- `easing?: TransformDefinitions.EasingDefinition` - 缓动函数，参见[TransformDefinitions](../../types/TransformDefinitions).[EasingDefinition](../../types/TransformDefinitions#EasingDefinition)
