在 NarraLeaf-React 中，过渡可以用于特定元素以创建视觉效果。

## 示例

```tsx
import { Dissolve } from "narraleaf-react";
```

```tsx
image.char("image1.jpg", new Dissolve(3000, "linear"));
```
