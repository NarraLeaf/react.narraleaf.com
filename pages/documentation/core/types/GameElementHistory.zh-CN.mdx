# GameElementHistory

```typescript
type GameElementHistory =
    | {
        type: "say";
        text: string;
        voice: string | null;
        character: string | null;
    }
    | {
        type: "menu";
        text: string;
        selected: string | null;
    };
```

`GameElementHistory` 类型表示可以记录在历史中的不同类型的游戏元素。它可以是：

1. "say" 元素：
   - `text`: 说出的文本
   - `voice`: 语音源 URL 或 null 如果未使用语音
   - `character`: 角色名称或 null 如果句子不是由角色说出


2. "menu" 元素：
   - `text`: 菜单文本
   - `selected`: 选中的选项或 null 如果菜单仍处于挂起状态
