# Font

```ts
type Font = {
    italic?: boolean;
    bold?: boolean;
    fontFamily?: React.CSSProperties["fontFamily"];
    fontSize?: React.CSSProperties["fontSize"];
};
```

## italic

Whether the text is italic.

## bold

Whether the text is bold.

## fontFamily

See [fontFamily](https://developer.mozilla.org/en-US/docs/Web/CSS/font-family).

## fontSize

See [fontSize](https://developer.mozilla.org/en-US/docs/Web/CSS/font-size).
