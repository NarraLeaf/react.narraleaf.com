# SentenceUserConfig

```typescript
type SentenceUserConfig = Partial<Omit<SentenceConfig, "voice"> & {
    voice: Sound | string | null | undefined;
}>;
```

For `SentenceConfig`, see [SentenceConfig](./SentenceConfig). That includes optional fields such as `pause`, `metadata`, and `avatar` (dialog avatars — see [Dialog](../../player/dialog)).
