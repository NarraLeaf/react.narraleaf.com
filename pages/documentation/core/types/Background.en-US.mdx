# background

```ts
type Background = {
    url: string;
} | color | null | undefined | StaticImageData;
```

The background of the scene. It can be a URL object, a [color](./color), a StaticImageData, or null.

> **Note:** StaticImageData is a type that generated by loader  
> If you are using NextJS, you can import image directly
> 
> ```ts
> import image from "path/to/image.png";
> ```
