# NarraLeaf-React Basic

Basic is a step-by-step tutorial that guides you from zero to a playable visual novel. You will learn how to create scenes, show dialogs, add choices, play sounds, and more.

If you are not yet familiar with **scenes**, **elements**, and **actions**, we recommend reading the [Core](/documentation/core) section first. It explains the core concepts of NarraLeaf-React.

## Prerequisites

Before starting this tutorial, make sure you have:

- Completed the [Quick Start](/documentation/quick-start) guide
- A working NarraLeaf-React project with `Player` and `GameProviders` set up

## Learning Path

The Basic tutorial is organized into the following steps. Follow them in order for the best experience:

| Step | Topic | Description |
|------|-------|-------------|
| 1 | [Create a Scene](/documentation/basic/create-scene) | Define a scene with background |
| 2 | [Add Actions](/documentation/basic/add-actions) | Add actions to make the story interactive |
| 3 | [Show Dialog](/documentation/basic/show-dialog) | Display character dialogue |
| 4 | [Show Images](/documentation/basic/show-image) | Display character sprites and images |
| 5 | [Play the Story](/documentation/basic/play-story) | Run your story in the React app |
| 6 | [Make Choices](/documentation/basic/make-choices) | Add branching choices for the player |
| 7 | [Play Sounds](/documentation/basic/sound) | Add background music and sound effects |
| 8 | [Store Data](/documentation/basic/store-data) | Persist game state and flags |
| 9 | [Conditional](/documentation/basic/conditional) | Create conditional branches |
| 10 | [Play Voices](/documentation/basic/voice) | Add character voice lines |
| 11 | [Manage Preferences](/documentation/basic/manage-preferences) | Handle game settings |
