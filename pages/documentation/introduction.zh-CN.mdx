# 介绍

欢迎来到 **NarraLeaf-React** 库的文档！

NarraLeaf-React 是一个轻量级、可定制且易于使用的 React 视觉小说框架。它旨在帮助您使用 React 创建自己的视觉小说。

## 为什么使用 NarraLeaf-React?

- **轻量级**：没有渲染库，没有不必要的依赖
- **可定制**：为您的应用程序定制一切，无需学习新语言
- **为开发者而设计**：基于面向对象编程原则，易于扩展和维护

NarraLeaf-React 取代了需要新脚本语言的传统引擎，提供了更灵活的解决方案。使用 **React** 和 **TypeScript**，您可以完全定制您的视觉小说的 UI，使其与您想要的一样独特——无需复杂的脚本。
