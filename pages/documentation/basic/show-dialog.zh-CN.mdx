# 显示对话

要显示对话框，你可以使用 [Character](../core/elements/character) 元素 

```typescript
import { Scene, Character } from "narraleaf-react";
```

```typescript
// 创建一个场景
const scene1 = new Scene("场景 1 - 醒来", {
    // ...
});

// 创建一个角色
const character1 = new Character("约翰·史密斯");

// 向场景添加动作
scene1.action([
    character1
        .say`你好，世界！`
        .say`早上好！`,
]);
```

名为 "约翰·史密斯" 的角色将会说 "你好，世界！" 和 "早上好！"。

你可以使用 [Word](../core/elements/word) 和 [Sentence](../core/elements/sentence) 元素为对话框添加一些自定义内容。

```typescript
character1.say`你好，我的名字是 ${Word.bold("约翰")}！`; // 将输出 "你好，我的名字是 约翰！" 并且 "约翰" 以粗体显示
```

你还可以使用这些元素构建复杂的对话框。

```typescript
const timeWord = new Word(
    (ctx) => new Date().toLocaleTimeString(),
    { color: "red" }
);

/* ... */

character1.say`现在的时间是 ${timeWord}`; // 将会输出 "现在的时间是 12:00:00 PM" 并且 "12:00:00 PM" 以红色显示
```

有关构建句子的更多信息，你可以阅读 [Sentence](../core/elements/sentence) 文档。

在这个例子中，角色将会说 "现在是" 并以红色显示当前时间。

你可以在文档中阅读更多关于 [Character](../core/elements/character) 元素的信息。
